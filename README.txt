
ЗАДАНИЕ:

   Обработать  информацию  о  фирмах  городов.  Обработку 
прекратить, когда на запрос будет введено слово «end». Фирм в городе 
не более пятидесяти. Информация следующая: 
    - название фирмы (не более тридцати знаков); 
    - величина налогообложения (не более 1 млн. р. – в виде строки); 
    - дата (месяц – в виде строки) последнего срока внесения налога; 
    - дата его фактического внесения (строка). 
В  одной  функции  внести  названия  фирм,  в  другой – величину 
налога,  в  третьей – предельную  дату  внесения  налога  и  дату,  когда 
налог погашен (если не внесен, то вводится нуль). В головном модуле 
для заданной даты (месяц) вывести в алфавитном порядке пять фирм, 
имеющих  максимальную  задолженность.  Глобальные  переменные  не  использовать. 
Исходный  массив  сохранить,  новых  массивов  структур  не  создавать. 
Можно объявлять и вводить другую необходимую информацию. 

Инструкция по использованию:
1.Запустите программу ( cd диск:/место/расположения/файла/ -> make -> ./lab1 ).
2. НАЗВАНИЯ ФИРМ. На предожение ввести название фирмы введите название (не больше 30 символов), 
затем жмём <enter>. Ввод прекратится когда вы введёте 50 фирм, либо если вам нужна остановка раньше, то
просто введите "end" (без кавычек).Тогда вы перейдёте к вводу персональной информации 
для каждой фирмы.
3. ПЕРСОНАЛЬНАЯ ИНФОРМАЦИЯ ПО КАЖДОЙ ИЗ ФИРМ.
    3.1. На предложение ввода налогообложения (tax), введите число (не больше 7 цифр).
    3.2. Далее на предложение ввести дату последнего срока ( deadline ), введите дату в
        формате DD MMM YYYY (Напр. 01 jan 1970) или запишите месяц при вводе полностью 
        (Напр. 01 january 1970).В случае неправильного ввода вам программа сообщит об этом.
    3.3. Далее на предложение ввести дату фактической выплаты, введите дату (см. пункт 3.2)
    либо "0" (ноль), если оплата налога не была произведена.
Пункты 3.1 - 3.2 повторяете для всех введённых фирм.
4. ДАТА, НА МОМЕНТ КОТОРОЙ БУДЕТ ПРОИЗВЕДЕНО ОТСЛЕЖИВАНИЕ ДОЛЖНИКОВ . На предложение ввести
дату мониторинга должников (monitor_date), введите дату в указаном формате (см. пункт 3.2).
