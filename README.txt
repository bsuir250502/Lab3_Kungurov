
ЗАДАНИЕ:

Реализовать программу учета занятых и свободных мест в общежитии, а также лиц, 
нуждающихся в общежитии. Одним из элементов созданной очереди является номер этажа, 
другим – общее количество мест на этаже, количество свободных мест, список лиц, проживающих на этаже.
Если свободных мест нет, то студент ставится в очередь на вселение на соответствующий этаж.
   

Инструкция по использованию:
1.Открыть test.txt
2.Ввести данный в следующем виде (для трёх этажей):
этаж
количество мест
свободных мест
список студентов (каждая фамилия на новой строке.Если фамилий меньше чем свободных мест, то в конце вводится "end" )
этаж
количество мест
свободных мест
список студентов (каждая фамилия на новой строке.Если фамилий меньше чем свободных мест, то в конце вводится "end" )
этаж
количество мест
свободных мест
список студентов (каждая фамилия на новой строке.Если фамилий меньше чем свободных мест, то в конце вводится "end" )
список студентов на заселение (каждая фамилия на новой строке.Если фамилий меньше чем свободных мест, то в конце вводится "end" )
3.Сохранить и запустить программу.
